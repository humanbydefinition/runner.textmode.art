# Interface: WebmExportOptions

Options for WebM exports.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-filename"></a> `filename?` | `string` | Suggested filename without requiring the runner to initiate a download. |
| <a id="property-framecount"></a> `frameCount?` | `number` | Number of frames to record. |
| <a id="property-framerate"></a> `frameRate?` | `number` | Capture frame rate. |
| <a id="property-quality"></a> `quality?` | `number` | Encoder quality hint. |
| <a id="property-transparent"></a> `transparent?` | `boolean` | Whether the export should preserve transparency when supported. |
