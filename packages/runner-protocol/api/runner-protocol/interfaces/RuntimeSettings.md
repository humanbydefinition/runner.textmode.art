# Interface: RuntimeSettings

Fixed runtime dimensions and timing used by configurable runner hosts.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-width"></a> `width` | `number` | Canvas width in CSS pixels. |
| <a id="property-height"></a> `height` | `number` | Canvas height in CSS pixels. |
| <a id="property-fontsize"></a> `fontSize` | `number` | Textmode font size in CSS pixels. |
| <a id="property-framerate"></a> `frameRate` | `number` | Target animation frame rate. |
