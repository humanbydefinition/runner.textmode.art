# Interface: SvgExportOptions

Options for SVG exports.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-includebackgroundrectangles"></a> `includeBackgroundRectangles?` | `boolean` | Whether to include background rectangles in the SVG output. |
| <a id="property-drawmode"></a> `drawMode?` | `"fill"` \| `"stroke"` | SVG drawing mode for text geometry. |
| <a id="property-strokewidth"></a> `strokeWidth?` | `number` | Stroke width used when `drawMode` is `stroke`. |
